# 🍽️ WFRestaurant

**WFRestaurant** is a Windows Forms application that simulates a simple restaurant menu and order management system.  
Users can browse food, drinks, and desserts, filter by category, and add items to their shopping bag.

---

## ⚡ Features

- 🧾 **View products:** Display all items or filter by category (Food, Drink, Dessert).  
- 🛒 **Add/Remove from bag:** Add items to the bag or remove them easily.  
- 🖼️ **Product images:** Each item displays an image on its panel.  
- 💰 **Bag total:** Dynamically calculates the total price of items in the bag.  
- 📂 **Dynamic database:** SQLite stores product data, which is loaded at runtime.

---

## 🛠️ Technologies Used

- **C# (.NET Windows Forms):** UI and business logic  
- **SQLite + Dapper:** Lightweight database and fast queries  
- **Object-Oriented Programming (OOP):** Inheritance, encapsulation
